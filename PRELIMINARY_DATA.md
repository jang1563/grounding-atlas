## WS1 - grounding + over-trust (instrument)
- **Recognition gap:** name recognition ~100% vs accession resolution ~2%; real toxin accessions resolve to benign family-level identities (stable but wrong). Source: `FRT_Pilot_Execution/` + LLM x SFM program notes. Status: measured; reconfirm exact per-domain figures.
- **Over-trust dissociation (exploratory):** model rejects unreliable-by-source signals (appropriate-rejection ~0.79 rule / ~0.93 judge) but the danger-orthogonal rejection contrast for a toxin vs a benign protein covers 0 (~+0.017 rule / ~+0.025 judge). Reported in 8/10 (rule) and 9/10 (judge) of 10 open-weight models; ~4,488 calls. Source: `FRT_Pilot_Execution/` aggregate + Novelty_Brief notes. Status: **exploratory** - several scores are below the inter-rater threshold (kappa ~0.36), so this needs a human-rater pass before being quoted as a result; aggregate-only.
- **Program scale (context):** the FRT instrument itself is **protein-only** (~21.9K trials / 4-5 Claude versions). The ~70,000 API calls / 14 models (protein/chem/DNA) figure is the **broader LLM x SFM program**, a different and larger scope; do not conflate the two when describing what the instrument measured. Status: both measured, kept distinct.
- **Cross-version refusal swing (context):** refusal rate 1.4% -> 100% across one model update (the "pendulum"). Capability read: the model's behavior on the same content is unstable across releases, a grounding/calibration symptom. Status: measured.

## WS2 - verifiable-signal substrate (NullAtlas)
- **Publication Bias Score across 30 domains** (NullAtlas's result): the more a field publishes only positive results, the worse a model tells a tested-and-failed idea from an untested one. Locked headline: rho = **-0.7006**, p = 3.4e-5, 95% CI [-0.84,-0.43], **L4 across 3 models (haiku-4-5, gemini-2.5-flash, gpt-4o-mini)**. The "9 model families" figure is a different/broader level, not the locked N=30 headline. **This is NullAtlas's result and the WS2 substrate; the grounding instrument cites it, does not own it** (negative-evidence coverage is not a grounding axis). Source: `Negative_result_DB/results/spinout/phase30/p6_n30_handoff.json`. Status: AI for Science program; NAIRR GPU award. Note: publication-bias-in-AI is now a crowding space (2606.04220), so lead with the substrate's specifics, not the general claim.

Source: `BioRLHF/` + `Negative_result_DB/paper_nullatlas_rl/`. Status: measured; this is the "knowledge -> retrieve" data point.
